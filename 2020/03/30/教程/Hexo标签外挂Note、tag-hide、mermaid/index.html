tag-hide

如果你想把一些文字、内容隐藏起来，并提供按钮让用户点击显示。可以使用这个标签外挂。

写法：

inline 在文本里面添加按钮隐藏内容，只限文字

( content 不能包含当引号，可用 ')

  • content: 文本内容
  • display: 按钮显示的文字 (可选)
  • bg: 按钮的背景颜色 (可选)
  • color: 按钮文字的颜色 (可选)

block 独立的 block 隐藏内容，可以隐藏很多内容，包括图片，代码块等等
