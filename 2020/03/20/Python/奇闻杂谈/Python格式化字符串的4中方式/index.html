五：总结四种方式的应用场景

1、如果格式化的字符串是由用户输入的，那么基于安全性考虑，推荐使用Template

2、如果使用的python3.6+版本的解释器，推荐使用f-Stings

3、如果要兼容python2.x版本的python解释器，推荐使用str.format

4、如果不是测试的代码，不推荐使用%
